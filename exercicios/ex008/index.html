Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (nao semantica)

Nesta frase, tenho um termo em destaque usando a tag <strong> (semantica)

Itálico / Ênfase

Nesta frase, temos um termo em italico usando a tag <I> (nao semantico)

Nesta frase, temos um termo em ênfase usando a tag <EM> (semantica)

Texto Marcado

Podemos criar tambem um texto marcado usando a tag <MARK>

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo com a tag <BIG> (nao semantica quase em desuso) e tag <small>

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas nao considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a a tag <ins>

Exixte tambem o sublinhado com tag <U> que é nao semantica

Texto Sobrescrito

para inserir coisa do tipo x20+3 como a tag <sup>

Texto Subescrito

para inserir coisas do tipo H2O com a tag <sub>
